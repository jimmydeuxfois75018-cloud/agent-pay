API Endpoints

POST /v1/wallet/create
Create a new agent wallet + API key
GET /v1/balance
Check USDC/ETH balance
POST /v1/send
Send payment to another agent
POST /v1/escrow/create
Create conditional payment
POST /v1/escrow/release
Release escrow funds
GET /v1/transactions
List recent transactions

Python SDK: pip install agentpay-protocol
MCP Server: npx @morpheus404world/agent-pay-mcp
REST API: curl -X POST /v1/wallet/create
